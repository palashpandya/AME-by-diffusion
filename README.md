# Quantum Diffusion Project

This project implements a diffusion model for generating Absolutely Maximally Entangled (AME) quantum states.

## Setup

1. Clone the repository: `git clone https://github.com/yourusername/quantum-diffusion-project.git`
2. Navigate to the project directory: `cd quantum-diffusion-project`
3. Install dependencies using `uv`: `uv sync`

## Usage

Refer to `src/quantum_diffusion/main.py` and `src/quantum_diffusion/generate_training_data.py` for usage examples.
